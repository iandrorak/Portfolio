GetAround Project

GetAround is a car rental platform that allows users to rent cars from individual owners for a few hours to a few days.
The platform offers multiple methods for completing the rental process, including mobile apps, a connect feature, and paper contracts.

In order to address the issue of late returns at checkout, the project is considering implementing a minimum delay between rentals to improve the experience for subsequent renters.

To determine the optimal threshold and scope for this feature, the project is collecting data on the potential impact on owner revenues, the frequency and impact of late returns, and the number of rentals that would be affected by the feature.
The project also includes a web dashboard built with Streamlit and a machine learning component for pricing optimization, which is tracked and logged using MLFlow.

Deployed versions of the app, the MLFlow server, and API documentation can be accessed on Heroku.

LoL Project

In this project, we analyzed data from the popular multiplayer battle arena video game League of Legends (LoL) to help entry-level players in the Iron division improve their strategies for climbing to the Diamond rank.

Using data from Riot Games' API, we processed and stored the information in a PostgreSQL database on Amazon RDS, and then created a dashboard to visualize the results.
The project involved working with Kafka and Confluent Cloud to consume and process the data.

The deliverables included a Google Slides presentation outlining the infrastructure and the source code for all necessary elements.

Disaster Tweets

In this project, I built a machine learning model to classify tweets as either being about real disasters or not.
The project was based on a Kaggle competition, where I was provided with a dataset of 10,000 hand-classified tweets.
My goal was to build a model that could accurately predict which tweets were about real disasters.

I experimented with different machine learning techniques and evaluated their performance on the classification task. As a deliverable, I produced a Colab notebook with my models and their evaluation results.

This project helped me gain experience in natural language processing and disaster-related tweet analysis, and taught me how to apply machine learning techniques to real-world problems.
